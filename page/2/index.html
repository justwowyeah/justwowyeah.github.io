人与人的差距和自我疗愈

发表于 2023-09-23 分类于 日常 ， 思考 本文字数： 351 阅读时长 ≈ 1 分钟

已是开学第三周的尾巴。漫长的时间，仍感怀时间飞逝。感知是相对的，运动速度和时间快慢密切相关。我不喜欢物理，只对故事和神秘主义感兴趣。相同的时间尺度下，进入心灵的事物越纷繁时间消逝地越快。睡梦同样，越美丽悠长越是惊叹几分钟光阴的伟大创造。相反，一整晚空洞的漫游怎么也弄不清是如何失去意识的。

我经验的就是这故事，现实的梦。

生活巨变。软工的课表不比从前复杂，但任务，不，工作量急剧增大。用这个词纯粹因为它们就是如此。之后两周我意识到，不仅仅是量变，更是生活和意识的质变：不再是学生，而是社会的一员，位置取决于贡献。

为变化苦恼时，另一种人在生活中存在。淡然处之，轻松完成，热情洋溢。认识并接受是另一种生活的开始。拒绝的同时拼命抓住什么来安慰自己。第一时间想到《小王子》。你的玫瑰和其他千万朵玫瑰不同，它是你的，独一无二。生活尽管平平无奇，才能和努力尽管不出众，不过，这是我的，我所拥有的，我宝贵的一切。

美学原理的第一问

发表于 2023-09-10 分类于 日常 ， 思考 本文字数： 927 阅读时长 ≈ 3 分钟

桌上的台灯亮着醒目的白光，我浸没在灯光里，思绪反复凝固活络。在家里玩闹了月余，消耗掉疲惫的两天把自己从中抽离，安顿在另一个地方的躯体里。回学校一个星期，我并没有一点分裂的感觉，似乎学习和放假完全是一回事。不论实质还是表象，大学已是和中学截然不同的另一模样，而小学却和中学逐渐靠近。

概率论、软件工程、web前端、编程实践……这些都是顺理成章的事情，远没有能让我写点什么的。因此，我要讲的是另一样东西：美学原理。令我意外的是，授课老师在b站有几十万粉丝，视频播放量也达到千万。尽管我的庆幸并非出自这一点。

高中毕业时，我和语文老师见过短短一面，他对我抱有小小的期望，劝我在学习之余不要忘了还有文学和艺术的世界在等着你。的确，纯粹的数理学习只会让我头昏脑胀，即使我再怎么热爱计算机，作为一部机器，它的冰冷和僵硬也仍然偶尔带给我眩晕。这种病入膏肓无药可医的状态，只有心灵的触碰能挽救。

“美学”是哲学的一个分支，亦或许“哲学”是美学的分支。这些都无关紧要，我要做的只是暂时忘记，只是思索。第一次课结束后，我拿到了问题：

如果世界上没有人，那么还会存在美吗？

如同我一直以来的考量，我率直地给出了否的意见。“美”和其它一切概念一样，不过是人给万物起的名字。实际上，万物只是简单地存在，它并不会选择成为什么，它只是恰好如此而已。区别出“美”和“丑”、“善”和“恶”，也只是认识世界而不得不行的办法，虽然很多时候是作茧自缚，但的确行之有效。

后来细想之下我却不这么认为了。我的思考太过以人类为中心了，我忽略了动物、植物，生命体，有意识的一切，他们都有一套自己的美。我能否认这一点吗？我能武断地认定只有人定义的美才是美吗？我不能。即使人从未存在过，美也照样存在。

然而很快我就发现，只要更深地剥离开人的存在，与答案的距离就会更近。“美”是人的主观概念，这毫无问题，“善”与“恶”也是人的主观概念。假如没有人来做观众，“善”与“恶”还会存在吗？我想是的，还会存在，“美”也会存在，并且是永恒存在。

为什么会这样？原因是同一句话：“‘美’是人的主观概念”，是人用来指代某种东西的名称。那么被指之物一定实实在在，无论是否指代它，无论是否有人，这东西都安安静静地在那儿，毫不意外。

恍然间，我以为我找到了真理，然而有声音要我冷静。它要求我仔细审视我的逻辑，剖析我的内心，然后返回到永远的思考，不可轻易地挣脱。世上有答案的事很少很少，多的是永无止境的争论，或许答案并非重要的东西，追求答案才是踏上旅途的原因。

因为确定的世界是无趣的，所以我会抱着思索和疑问迎接每一个明天。

如何看待事物

发表于 2023-08-28 分类于 日常 ， 思考 本文字数： 765 阅读时长 ≈ 3 分钟

近来散步，今天被路边老人称为“好孩子”，我不禁哑然，何以散步竟成了好孩子呢？

要琢磨透这件事，就得来看看我们是如何评价一个人的。评价人的尺子继承自评价万物的规则。各人心中有一把尺子，世上的一切事物来到他面前接受他的度量，然后在他的心里分出个好坏高低来。正因人的尺子各不相同，于是万物的好坏高低各有说法，在人群之中我们不得不屈服于他人，但是，在乡间散步的道路上我们仍是可以多少遵从内心的指引的。

事情显然是这样，我每日散步经过老人家门前，今日他总算把对我的评判讲了出来。评价总是过去式的，它在现时发出，对未来造成影响。通常很难意识到这一点：评价的归因是过去。人们反而总是将它与现时紧紧联系起来，尤其是与具体的绝对的事物联系起来，而不是与事物的运动变化。事实上，我们很难对永远变化的东西做出任何评判，只能观察历史中的某些片段得出短暂的结论。

然而，历史的片段具有这样一个特性：一个截面上的一切是从上一个瞬间脱胎。“大道五十，天衍四九，人遁其一。”在每一个瞬间，人能付出的努力至多改变五十之一的可能，而余下的都由人以外的东西抉择。我们来看看这是些什么：首先是某种规则，它潜藏在万物本质之下，高出于世间一切真理；接着是一点不确定性，它是由物理法则深深钳制的；最后是人过往的努力。是的，在某一截面以前，能动的人付出的一切都参与进了对将来的审判。无怪乎许多人会感到无力，是他们自己将自己拉扯在原地。

现在我们明了这一切了，过去的幻影不断击打在一个人的内心，这个人不得不好好对它，把它收拾在某个做了标记的抽屉里，这份标记可能是善，可能是恶，也可能是别的什么东西，不管怎样，这是由尺子测量所得。

那么，回到正题。老人对我做出“好孩子”的评价，在我看来是不能接受的，对我的尺子而言，散步不能属于“好”的原因，而且正因为评价的短暂和主观性，通常我很难开口说什么，尽管心里有无数话，但都不是能说出来的话。然而，我不会对这一切做什么纠正，有什么好反驳或赞同的呢？人的尺子不同，各种事物有着不同的模样，这就是一个事实，只等待你的接受。

理想主义的本质

发表于 2023-08-27 分类于 日常 ， 思考 本文字数： 503 阅读时长 ≈ 2 分钟

《人物》公众号的一篇文章《被分成两半的女孩》很让我感触，暂未读完，写的是创新教育模式塑造的人与另一个世界的撕裂感。其中有这么一段：

一个事物的正面和背面总是连在一起。森林学校试图提供一个理想的世界，但缺点在于，理想主义是一种乌托邦，它构造的是一个尚未到来的世界。

这番话令我十分感慨，正巧教育模式也是我很感兴趣的话题，所以读来倍感有趣。

“理想主义是一种乌托邦”，它的根基是虚浮的，犹如空中阁楼。“它构造的是一个尚未到来的世界”，更不如说，是一个永远不会到来的世界，正如我曾经说过的一句话：理想从来不是用来达到的，而是用来追求的。——这是我发在b站一个视频《一个理想主义者的自杀》里的一句弹幕，视频很好，可以去看看。

说理想世界永远不会到来并不是我的悲观和消极，恰恰相反，多数时候我是一个积极无比的理想主义者，坚信我所信奉的就是真理。但是真理毕竟是局限的，何况理想世界的塑造只是真理的投射罢了，它并非永恒常在，而是不断变化，随着社会的发展而改变它的样貌与内核，只不过它遵守的规则永不改变。

接受理想主义的空虚根基，信仰理想世界的同时觉悟自己的终身追求不过海市蜃楼，正确地认识到自己是一个真正的现实主义者。这是理想主义者的最高标准。心灰意冷者，是还没有意识到自己追逐的火光背后模糊的虚影是什么。

为理想主义高歌，理想主义者万岁。

重新开始——我怎么对待博客？

发表于 2023-08-27 分类于 学习 ， 日常 ， 博客 ， 思考 本文字数： 282 阅读时长 ≈ 1 分钟

继六月建立博客、七月写下最后一个字以来，我在直到八月末尾的时光里虚掷了许多时间与精力，完全彻底地抛弃了这个来之不易的孩子。

现在我决定重新开始，但是需要转换思路。

之前我雄心勃勃地试图把这个地方建立成我的思想王国、技术殿堂、人生归宿，但是我高估了自己的专注力，同时也低估了写长文的难度。许多洋洋洒洒的计划只是进行了开头就丧失了兴趣，转而投入到下一个选题之中，最后什么也没剩下。

现在我比两个月前更有认知，尽管我仍有一个长文技术贴的梦，但还是先把这个梦放在未来吧，现在我只需要写一点简单的东西，让自己的表达落到具体的地方，而不是空气里。

一句话，我在这里发微博，与那个人尽皆知的微博所不同的是，这里没有付费，没有花哨，没有人。

岁月失语，唯石能言

发表于 2023-07-06 分类于 日常 ， 思考 本文字数： 632 阅读时长 ≈ 2 分钟

因小见大，见微知著。这是人认识世界的重要办法。人无法达到真理的高度，却能无限逼近真理。

人对自身发生思考，对身边的事物发生思考，这些思考经过推广、抽象、上升，形成对宇宙的思想。

尽管这个过程有着逻辑认知保障，但仍然免不了犯错。错误发生在自身，于是自身毁灭；发生在对外物的认识，于是外物受损；发生在对真理的逼近，于是思想崩溃。

许多渺小的思考是受到忽略的，譬如战争时的一扇旁门、公式里的一处不和谐……然而它们偶尔能抵达真理。

我们常常只能首先做渺小的思考，这是出于自身渺小的缘故。我近日幸而得了这样一个契机。

我的蓝牙耳机返厂维修获得全额退款，固然是件好事，然而我终于有些留念与不舍。毕竟它是陪了我半年的老朋友，当我观摩时，它的质感令我出神。

世间永无不散之宴席。假若有幸，数年之后于电子市场仍能撞见当年朋友，重又结一段缘分。仿佛《但是还有书籍》所记录的淘书人，得书、览书、卖书，复得之。此番旅程凝固了太多时间和故事，闲暇时细读书中字迹，摩挲扉页折痕，足以消遣尽兴。

然而硬币的另一面却使人忧郁：倘使我与老友再无会面之期，又该借何种办法抒解心底郁结？

我所以抒解心绪之物，必然常在我身旁，否则我不能抵达；它又必然能伴我度过余生，否则我今日得以舒缓心情，明日却又重复忧郁；其更要永恒常在，于思想能潜入的最深远处仍要熠熠闪光，足以照耀我顽固探索的脚步之下的径道。

我暂不能明白这一切。但是倘若要我给人生和世界上色，我不会不涂抹上一笔跃动的色彩：因为我是不能苟活在单调乏味的生活里的。我既然已经活，并且决定继续活，那么这生活不能不好好回应我。

然而，承认存在某种坚固的东西，这难免不是另一番令人发怵的想象。

思考至此，思路全无，遂投笔离席。

Hexo使用报告

发表于 2023-06-25 分类于 学习 ， 博客 本文字数： 503 阅读时长 ≈ 2 分钟

Hexo使用文档告诉我们：

你可以执行下列命令来创建一篇新文章或者新的页面。

写作 |Hexo

  • 首位的 $ 是 Git Bash 自动显示的，不是你应该输入的内容

因此，当你想要写一篇题为 “第一篇博客” 的草稿时，你可能会尝试：hexo new [draft] <第一篇博客>
但你不会收到任何效果，因为这种做法实际上是错误的。你真正应该敲下的是：hexo new draft 第一篇博客

不仅如此，在后文中你会见到这么一个符号，它频繁出现，但是通常是可以被忽略的：""
这是 “” 在代码框中的样式，它不同寻常，不是吗？

由此，我们得知了第一个重要的规则：

[ ]、<>、""通常不是输入内容，而是提示输入的存在。

在阅读 Hexo使用文档 的时候，这一点是不得不记住的，因为你很可能会不断地栽在这上面。试想：一篇帮助文档的第一句话（实际为基本操作的第一句话）就隐含了如此巨大的未知信息，它显然把真正的初学者看得太聪明了，如果你不幸正欲从此入门，那么阅读本文是你的不二之选。
